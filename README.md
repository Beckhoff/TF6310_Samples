# About this repository
This repository contains official sample code for the product TF6310|TC3 TCP/IP offered by [Beckhoff Automation](https://www.beckhoff.com). The sample code is provided as-is under the Zero-Clause BSD license.

## Samples overview

| Category | Sample | Description |
| --- | --- | --- |
| Container | [container-samples](container-samples/README.md) | Overview of container-based TF6310 samples for TwinCAT 3.1 XAR deployment with Docker. |
| C# | [SampleClient](C%23/SampleClient/README.md) | TCP/UDP sample client for connect/send/receive workflows. |
| PLC / Getting Started | [GettingStarted](PLC/GettingStarted/README.md) | Introductory PLC sample with local TCP client/server communication. |
| PLC / TCP | [Sample01](PLC/TCP/Sample01/README.md) | Basic TCP client and server PLC projects. |
| PLC / TCP | [Sample02](PLC/TCP/Sample02/README.md) | Single-client/single-server string communication with TcSocketHelper. |
| PLC / TCP | [Sample03](PLC/TCP/Sample03/README.md) | Multi-client string communication against a PLC server. |
| PLC / TCP | [Sample04](PLC/TCP/Sample04/README.md) | Single-client/single-server binary communication with TcSocketHelper. |
| PLC / TCP | [Sample05](PLC/TCP/Sample05/README.md) | Multi-client binary communication against a PLC server. |
| PLC / TCP | [Sample06](PLC/TCP/Sample06/README.md) | TLS-secured TCP client/server sample with certificate-based setup. |
| PLC / TCP | [Sample07](PLC/TCP/Sample07/README.md) | TLS-secured TCP client/server sample with PSK-based setup. |
| PLC / UDP | [Sample01](PLC/UDP/Sample01/README.md) | UDP peer-to-peer communication using two PLC peers. |
| PLC / UDP | [Sample02](PLC/UDP/Sample02/README.md) | UDP multicast communication between two PLC devices. |
